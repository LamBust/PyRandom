Библиотека random — генерация случайных чисел и элементов

Библиотека random используется для работы со случайными значениями. Она позволяет добавлять элемент случайности в программы, что особенно важно при создании игр, симуляций, тестов и учебных примеров. Все значения генерируются псевдослучайным образом, то есть на основе алгоритма.

С помощью random можно получать случайные числа в заданном диапазоне, выбирать случайные элементы из списков, а также перемешивать последовательности данных. Это делает библиотеку удобной для моделирования случайных событий.

Пример использования:

import random

print(random.random()) # случайное число от 0 до 1
print(random.randint(1, 100)) # случайное целое число
names = ["Анна", "Иван", "Олег", "Мария"]
print(random.choice(names)) # случайное имя
random.shuffle(names) # перемешивание списка
print(names)
